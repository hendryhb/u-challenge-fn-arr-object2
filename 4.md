# Sort Alphabet

## Objectives
- Able to solve a given problem
- Able to sort characters other than number type characters

### RESTRICTION

- Only allowed to use built-in function `.push()`

### HINTS

- The function name must be `sortAlphabet` and __cannot be changed to any other function name__. For function details, refer to [Directions](#directions) below

## Directions

Given a function *sortAlphabet*() that receives one parameter in the form of a string.

The function will return a new string that has been sorted alphabetically from a to z.
For example, the string 'hallo' will become 'ahllo', or the string 'num' will become 'mnu'.
You do not need to consider if there are symbols or numbers in the string.

```js
function sortAlphabet(str) {
  // you can only write your code here!
}

// TEST CASES
console.log(sortAlphabet('hello')); // 'ehllo'
console.log(sortAlphabet('truncate')); // 'acenrttu'
console.log(sortAlphabet('developer')); // 'deeeloprv'
console.log(sortAlphabet('software')); // 'aeforstw'
console.log(sortAlphabet('aegis')); // 'aegis'
```
