# Logic Challenge - Sorting!

## Objectives

- Able to create a function similar to simple built-in JavaScript functions
- Able to combine the concepts of `looping/iteration` and `conditional` in a problem
- Able to access an array
- Able to swap values of two variables

### RESTRICTION

- Only allowed to use built-in functions for adding or removing data in arrays, such as `shift()`, `unshift()`, `push()`, and `pop()` or built-in functions for changing characters to uppercase or lowercase.

### HINTS

- The function names must be `sorting`, `sortingByType` and `sortAllClean` and __cannot be changed to any other function names__. For function details, refer to [Directions](#directions) below

---

## Directions

Create a function that sorts an array from smallest to largest.

***NOTES*** Forbidden to use built-in function .sort

**Release 0**

Sort an array whose values are all of number type.

```js
function sorting(array) {
  // your code here
}

console.log(sorting([ 2, 4, 6, 8, 2, 3 ])); //[ 2, 2, 3, 4, 6, 8 ]
```

**Release 1**

To make it more interesting, you are asked to create sorting of the array based on the data type from the given array

```js
function sortingByType(array) {
  // your code here
}

console.log(sortingByType([ 1, 3, 'array', -45, true, false, 'big' ]));
// [ [ -45, 1, 3 ], [ 'array', 'big' ], [ false, true ] ]
```

**Release 2**

Next, you are asked to handle unwanted data such as null, undefined, NaN and empty array ([]) by removing those weird data from the array before sorting.

```js
function sortAllClean(array) {
  //your code here
}

console.log(sortAllClean([undefined, null, 456, 'def', NaN, [], true, 123, 'bcd', false]));
//[ [ 123, 456 ], [ 'bcd', 'def' ], [ false, true ] ]

console.log(sortAllClean([ NaN, undefined ])); // []
```
