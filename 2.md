- Understand the connection between one function and another

### RESTRICTION

- Only allowed to use built-in functions for adding or removing data in arrays, such as .shift(), .unshift(), .push(), and .pop()

<hero_name>-<hero_type>,<hero_name>-<hero_type>, ...

```
[
  [ <list_of_heroes_with_ranged_type> ],
  [ <list_of_heroes_with_melee_type> ]
]
```
If the input is an empty string ('') then return an empty array

To separate the hero name and type, implement the provided splitting() function where the result of the function is a one-dimensional array.
If given 'Razor-Ranged,Meepo-Melee,Invoker-Ranged'

Expected output from splitting():
```
['Razor-Ranged', 'Meepo-Melee', 'Invoker-Ranged']
```
